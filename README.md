# UNINUS GOLANG GRPC BOILERPLATE

# How to run ?
1. Clone this project with SSH or HTTPS
    `git clone <url>`

2. Create Database
    `docker-compose up -d`

3. Migrate table
    `make migrate-up`

4. Run Project
    `make run-local`

> If u want to edit config, go to `config/local.conf`
